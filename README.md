## 🌟 功能亮点

- **科学计算**  
  ✅ 三角函数（sin/cos/tan）  
  ✅ 对数/指数运算（log, ^）  
  ✅ 括号支持复杂表达式

- **现代 UI**  
  🌙/☀️ 实时主题切换（暗黑/浅色模式）  
  📋 计算历史记录（带点击恢复功能）  
  ⌨️ 完全键盘支持

- **智能特性**  
  🚨 自动错误处理  
  📊 响应式布局（自适应屏幕）  
  🔄 遵循数学运算优先级
